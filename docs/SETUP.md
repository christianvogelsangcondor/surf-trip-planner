# Setup Instructions for Surf Trip Planner

## Prerequisites
- Node.js (14.x or later)
- npm (6.x or later)

## Installation Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/christianvogelsangcondor/surf-trip-planner.git
   cd surf-trip-planner
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```

## Additional Configuration
- Make sure to configure your environment variables as specified in the `.env.example` file before starting the application.

## Running Tests
To run the tests, use:
```bash
npm test
```
